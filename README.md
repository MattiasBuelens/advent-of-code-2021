# Advent of Code 2021
My solutions for [Advent of Code 2021](https://adventofcode.com/2021/), written in [Rust](https://www.rust-lang.org/).

## Running
This project uses [cargo-aoc](https://github.com/gobanos/cargo-aoc) by [@gobanos](https://github.com/gobanos/).

To run the solution for a specific day (e.g. day 1):
```sh
$ cargo aoc -d 1
``` 

To run all solutions:
```sh
$ cargo run
```
